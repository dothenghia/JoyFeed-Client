# JoyFeed - Thiết bị cho thú cưng ăn từ xa

## 1. Giới thiệu
- Sản phẩm đồ án của nhóm 5 :
    + Đỗ Thế Nghĩa
    + Mạc Tuấn Trung
    + Nguyễn Cao Khôi

## 2. Chức năng

Web :
- Đăng kí
- Đăng nhập
- Đăng xuất
- Khôi phục mật khẩu
- Đổi mật khẩu
- Theo dõi lượng thức ăn hiện tại
- Cho ăn ngay lặp tức
- Cài đặt hẹn bữa ăn
- Xem lại lịch sử

Máy :
- Phát thức ăn
- Phát ra âm thanh
- Gửi thông báo khi thú bỏ ăn
- Gửi thông báo khi gần hết thức ăn (20%)
